# 🗓️ Meeting Minutes Generator

## Overview
Auto-generates meeting notes from transcripts.

## Tools
- Otter.ai (or Zoom)
- Zapier
- GPT-4
- Google Docs

## Workflow
1. Upload transcript.
2. Zapier fetches and sends to GPT.
3. GPT creates clean summary.
