# 📝 AI Blog Post Generator

## Overview
Generates long-form content based on a topic entered in a form.

## Tools
- Typeform
- Zapier
- GPT-4
- Google Docs

## Workflow
1. User inputs topic in Typeform.
2. Zapier sends topic to GPT.
3. GPT generates article.
4. Article saved to Google Docs.
